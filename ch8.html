Terms Definitions
1. Information Security Is the goal of keeping information protected from those who do not have permission to access it. Electronic information such as, data stored on computers hard disk, cloud, flash drive, or smartphone.
2. Authentication Is the process of verifying who has the right to gain access to the computer. Operatng systems maintains a file of users ID's and Passwords.
3. Authorization Governs what an authenticated user of a computer is allowed to do. For example, when you log into your bank account, you use Authentication to gain access, Authorization if having access to only your account and features but not able to change balance or interest rates.
4. Dual Authentication Is a second authentication measure after teh user enters the user ID and password. This second form of authentication can be a randomizced numbers sent to the users phone to enter, it can also be a CAPTCHA to ensure your a acutlly person logging on.
5. Social Engineering The process of using people to get the information you want, like a sensitive information or passwords. This can be either direct or indirect approach.
6. Virus A computer program that embeds itself within another program or file in order to infect a host computer and then spread.
7. Worm Similar to a virus, t can send copies of itself to other nodes on a computer network without havng to be carried by an infected host file.
8. Trojan Horse A computer program that does some legitimate computatoinal task but, unbeknownst to teh user, also contains code to perfrom malicious attacks.
9. Ransomware Malware that invades a user's computer, encrypts all files, then demands a ransom payment for teh files to be released.
10. Phishing The practice of sending emails purporting to be legimate to a large number of potential victims in the hopes of luring users to visit a fake website, where personal information can be collected. For exmaple, a link that could take you to your bank(which you thought was yours but fake) then steal your login credentials.
11. White Hat Are security experts who try to keep us safe in cyberspace.
12. Black Hat The people who try to ferret out computer system weaknesses for financial gain, control, or outright destruction.
13. Encryption/ Decryption A security measure in place to transmit data securely. It makes imofrmation meaningless incase someone manages to steal it. The process is done by using an algorithmn to convert information into a representation that cannot be understtod or utilized by anyone without the decryption algorithm. Decryption is the process of reversng teh effectts fo encryption back into its ornginal plaintext.
14. Crytography Science of "Secret Writing". Which can be refered to encryption and decryption. The message that is being encrypted is called ciphertext.
15. Symmetric Encryption Algorithm An encryption algorithm that requires a secret key shared by sender and receiver.
16. Caesar Cipher An encryption algorithmn that shifts each character in the message to another character some fixed distance farther along in the alphabet. This is also called shift cipher.
17. Block Cipher Any encryption algorithmn that encodes a block of characters together, so that each coded character is the result of several plaintext characters.
18. Data Encryption Standard (DES) Designed to protect electronic informaton, so the plantext is a binary string of 0s and 1s, just as it is stored in a computer. It is an encryption algorithm developed by IBM, and certifed as an international standard by the ISO.
19. Public Key System An asymetric encryption algorithm that uses two keys, a public key for encryption and a private key for decrypting them. It allows anyone to send a secure message to a reciever without needing to share a scret key, since the reciever will have the private key.
20. Embedded Computers Computational devices such as chips, processors, or computers that are embedded within another system. They could be found in: your car, home thermostat, cell phone, microwave, bank ATM, etc.
How to hide text files in a steganographic image: This can be achieved by changing the least significant bit of each byte of the image file to match the binary form of the characters in the text.
Six ways to protect yourself from most cyber attacks
  1. Ensure your computer has up-to-date aniivirus software. This helps dectect and remove worms, viruses, and trojan horses from your computer.
  2. Be sure to install or update your firewall software, to help guard your computers access points and blocks unauthorzied communication or operations.
  3. Install up-to-date antispyware software which routinely scans and removes spyware that may track your web activity, steal passwords or credit card information.
  4. Set up your browser to use a pop-up blocker, to prevent you from clicking on malicious pop-ups.
  5. Always install the latest security patches or updates to your operating system.
  6. Avoid sharing personal or financial information through suspicous emails.
